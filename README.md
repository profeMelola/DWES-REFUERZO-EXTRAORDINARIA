# Conceptos vistos en el curso

## Fundamentos vistos en clase

- Jakarta EE, Servlets, ciclo de vida HTTP
- Paso a Spring Boot — configuración, autoconfiguración

## Capa de datos

- JPA e Hibernate — entidades, relaciones OneToMany, ManyToOne, ManyToMany con tabla intermedia y atributos propios, claves compuestas con @EmbeddedId y @MapsId
- Cascade, orphanRemoval, FetchType
- Spring Data JPA — repositorios, derived queries, JPQL con proyecciones a DTOs
- Paginación y filtrado

## API REST

- Arquitectura en tres capas Controller → Service → Repository
- DTOs con mappers manuales
- Bean Validation
- Manejo global de excepciones con @RestControllerAdvice
- Códigos HTTP semánticos con ResponseEntity
- Enumerados como tipos de dominio

## Spring MVC

- Thymeleaf con fragments y layouts
- Patrón PRG con RedirectAttributes
- Consumo de API REST con WebClient

## Spring Security

- Autenticación con formulario
- Roles y autorización por URL y por vista
- @AuthenticationPrincipal
- BCryptPasswordEncoder
- JWT
